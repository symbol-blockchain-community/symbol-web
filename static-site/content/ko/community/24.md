## 다음은  NEM/Symbol 기반 Dapps 및 서비스 소개 페이지입니다.

### **Wallet**
#### `Symbol`
* [**Desktop wallet**](https://github.com/symbol/desktop-wallet/releases) Linux, Windows and Mac
* [**Mobile wallet** Android](https://play.google.com/store/apps/details?id=nem.group.symbol.wallet&hl=ja&gl=US) 
* [**Mobile wallet** iOS](https://apps.apple.com/us/app/symbol-wallet/id1528741845) 
* [**Mobile wallet AEM+** Android](https://play.google.com/store/apps/details?id=com.aem.plus) 
* [**Mobile wallet AEM+** iOS](https://apps.apple.com/jp/app/aem/id1607457067) 
AEM+ 지갑은 모든 기기에서 암호화폐 거래를 관리할 수 있는 멀티체인(XYM, XEM, BTC, ETH) 모바일 지갑입니다. 모든 암호화폐 지갑을 한 곳에 보관할 수 있도록 설계되어 손바닥 안에 있습니다. 
* [**Mobile wallet EXYM** Android](https://play.google.com/store/apps/details?id=com.ry0chang.EXYM) 
* [**Mobile wallet EXYM** iOS](https://apps.apple.com/jp/app/exym/id6449445030) 
EXYM 은 일상에서 디지털 아트를 쉽게 즐길 수 있는 새로운 스타일의 암호화폐 지갑 애플리케이션입니다. 사용자가 자신만의 스킨 디자인 지갑을 쉽게 만들 수 있어 기존의 NFT 사용을 뛰어넘는 새로운 경험을 선사합니다. 아트 크리에이터에게는 자신의 작품을 새로운 형태의 가치로 즐길 수 있는 기회를 제공합니다.
* [**Mobile wallet CoinPocket** Android](https://play.google.com/store/apps/details?id=net.teritaris.coinpocket&hl=ja&gl=US)
* [**Mobile wallet CoinPocket** iOS](https://apps.apple.com/jp/app/coinpocket/id6479215775)
코인포켓은 다른 암호화폐 지갑의 복잡함 없이 쉽게 사용할 수 있는 것을 목표로 합니다.
* [**Browser wallet NFT-DriveEX**](https://chrome.google.com/webstore/detail/nftdriveex/oodjpaddnaphopckiphmgapkafegcghl)
NFTDriveEX는 사용자가 사이트 어디에서나 NFT, XYM 및 기타 mosaics 를 교환할 수 있는 P2P 지갑입니다.
[Google Chrome Extension Version](https://chromewebstore.google.com/detail/nftdriveex/oodjpaddnaphopckiphmgapkafegcghl?hl=ja)
[Web(Universal version)](https://nftdrive-ex.net)
* [**Browser wallet SSS(Safely Sign Symbol) Extension**](https://chrome.google.com/webstore/detail/sss-extension/llildiojemakefgnhhkmiiffonembcan)
Symbol Blockchain SSS Extension 은 Symbol blockchain 을 사용하는 웹 애플리케이션에서 작동하는 Google Chrome Extension 입니다. 이 SSS 애플리케이션은 개인 키를 처리할 필요 없이 Symbol blockchain 에서 트랜잭션에 서명하도록 설계되었습니다.
이 앱은 거래에 안전하게 서명할 수 있는 확장 프로그램이지만, 브라우저 지갑으로도 사용할 수 있습니다.
* [**Browser wallet Ventus Wallet**](https://ventus-wallet.tk)
Google Chrome Extension "SSS Extension"을 사용하는  Symbol 웹 지갑입니다. 
* [**Browser wallet One Plate**](https://oneplate.food-nft.io)
ONEPLATE 는 NFT를 시작하기 위한 사용하기 쉬운 도구입니다. '누구나 지금 당장 할 수 있다'는 목표로 이해하기 쉽고 사용하기 쉽도록 설계되었습니다.

* [**QUEST**](https://quest-bc.com)
QUEST는 블록체인 기술을 기반으로 하는 새로운 '우발적 수수료 기반' 팁 플랫폼입니다.
"The quest"는 도전에서 목표로 하는 업적입니다.
여러분의 도전 여정과 성공적인 피날레를 기대하는 후원자들은 기부를 통해 퀘스트의 금고에 힘을 보탭니다.
The quest 를 완료하면, 금고의 내용물을 획득할 수 있습니다. 퀘스트를 완료하지 못하면 기부한 후원자에게 환불됩니다.
하지만 'the quest'의 성공과 실패는 누가 판단할까요? 바로 서포터들의 투표입니다.
"QUEST"의 주요 특징은 블록체인 기술로 구현된 완전 자동 및 안전한 처리입니다.
* [**Token Live**](https://tokenlive.me/#/)
Token Live 는 symbol blockchain mosaic (token) 를 활용한 라이브 스트리밍 서비스입니다.
배포자가 지정한 특정 NFT(unique) 또는 양도 불가능한 FT(bundle)를 소유한 사람만 참여할 수 있습니다.
* [**ichigo**](https://ichigo-word.fun)
Ichigo 는 symbol blockchain 을 사용하는 게시판 형식의 소셜 네트워크 미디어로, XYM 팁 기능이 있습니다.

-Image files：GIF, JPEG, PNG, SVG
-Audio files：MP4, MP3, MIDI
-Video files：MP4, AVI
-Other files：TXT, PDF, HTML(CSS Javascript), ZIP

UMUI는 오키나와 여행, 중요한 기념일, 결혼식, 소중한 가족과의 시간 등 오키나와에서의 추억을 Symbol public blockchain 에 사진, 동영상, 오디오 파일로 저장하는 서비스입니다.
-Image files：GIF, JPEG, PNG, SVG
-Audio files：MP4, MP3, MIDI
-Video files：MP4, AVI
-Other files：TXT, PDF, HTML(CSS Javascript), ZIP
