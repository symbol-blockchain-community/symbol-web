IPDC(IP DataCast; 인터넷에서 일반적으로 사용되는 IP 패킷을 방송 전파에 겹쳐서 동시에 배포하는 방송 서비스의 총칭)와  Symbol 을 이용한 재난 현장의 신뢰성 검증 시스템 블록체인은 KTV Software Inc.(Kansai Television Software Inc.) 의 "放送技術"(방송 기술), 76권, 11월호에 게재되었습니다.

[Announcement from KTV Software Inc.](https://www.ktvs.co.jp/news/231027.html)

방송 산업을 중심으로 다양한 기업용 시스템을 개발하는 이 회사는 블록체인 기술의 위변조 방지 기능과 넓은 지역에 걸쳐 안정적인 정보를 전송할 수 있는 방송 전파의 커버리지를 활용해 재난으로 통신이 두절되더라도 IPDC를 통해 정보의 신뢰성을 확보하는 것이 목표라고 설명했습니다.

Symbol 퍼블릭 블록체인의 채택은 보안과 사용 가능한 검증 기능으로 설명할 수 있습니다.

또한 이번 시연에서는 실제 방송 전파를 사용하지 않고 이론적 테스트와 유선 실험을 통해 시스템의 유효성을 검증했다고 밝혔습니다.

이 실험의 내용은 11월에 열리는 [Inter BEE 2023](https://www.inter-bee.com/en/) 에서 전문가들이 미디어의 가속화된 진화와 미래를 경험하고 교류할 수 있는 종합 미디어 이벤트에 전시될 예정입니다. 이 행사는 2023년 11월 15일 수요일부터 17일 금요일까지 Makuhari Messe 에서 개최됩니다. 이 작품은 전시 기간 내내 시연 실험에 협력한 [ATTRACTOR Co., Ltd.](https://www.att-star.co.jp/index.html) 의 부스에서 전시될 예정입니다.
