XYMcomic 는 Fully-On-Chain NFT의 만화 시리즈로 the official XYMcomic Twitter 에서 일본어와 영어로 2주마다 공개됩니다. 이 작품은 심볼 블록체인을 사용하여 풀 온 체인 NFT로 게시되고 Twitter 에서 경매됩니다.
 [the official XYMcomic Twitter](https://twitter.com/xym_comic?s=20)
 
 일러스트레이터 [Toujou-Mina 東城みな](https://toujoumina.com) 가 그린 만화를 [Atomos-Seed LLC](https://atomosseed.com) 가 [NFTDrive](https://nftdrive.net) 를 사용하여 Fully-On-Chain NFT 라는 업로더가 Symbol blockchain 를 사용하여 제작합니다.

* [Toujou-Mina 東城みな Twitter](https://twitter.com/toujoumina?s=20)　
* [Atomos-Seed LLC Twitter](https://twitter.com/atomos_seed?s=20)　
* [NFTDrive Twitter](https://twitter.com/nftdrive_jp?s=20)　

Fully on-chain NFT은 이미지와 같은 데이터가 Fully on-chain NFT 외부의 파일 시스템에 저장되는 일반적인 형태의 NFT가 아닌 모든 관련 데이터를 blockchain 에 직접 저장합니다. 이렇게 하면 데이터 자체의 손실이나 변조 위험이 줄어들고 데이터를 안전하게 저장할 수 있습니다. XYMcomic는 NEM/Symbol community "XYM CITY"를 중심으로 스토리가 전개되는 일련의 blockchain 만화입니다.

Fully On-chain NFT를 생성하기 위해 Fully On-chain NFT 완전 온체인 NFT 업로더 NFTDrive가 사용되며, 이는 데이터를 거의 영구적으로 저장할 수 있고 저작권을 명확하게 표시할 수 있음을 의미합니다.
 

이 페이지는 지금까지 공개된 모든 만화의 이미지를 모아놓은 아카이브입니다.(Episodes 1-20)
