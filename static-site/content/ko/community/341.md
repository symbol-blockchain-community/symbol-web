* [**NFTDrive**](https://nftdrive.net/?nav=nftdrive)

* [**NFTDriveEX**](https://nftdrive.net/?nav=ex)
  * 모든 사용자가 완전한 온체인 NFT를 P2P로 전송할 수 있도록 하는 것을 목표로 개발된 일본산 NFT 전용 지갑입니다.
* [**HAYABUSA**](https://nftdrive.net/?nav=hayabusa)
  * 완전한 온체인 생성 NFT 생성 도구. 임의의 지도 데이터와 이미지 데이터를 하나의 NFG 파일로 결합하는 단일 NFG 파일 방식을 개발하여 Symbol blockchain 에 단 한 번의 쓰기로 수천 개의 생성 데이터를 완료할 수 있으며 가스 요금도 절약할 수 있습니다. 프리민트나 세일민트도 가능하며, 민트사이트는 허용 목록과 공개 기능을 지원하며 EVM을 지원하는 대부분의 체인에서 사용할 수 있습니다. 사용자가 완전한 온체인 NFT를 생성할 필요가 없는 경우 GoogleDrive 내에서 오프체인 생성 NFT를 생성할 수도 있습니다.
* [**Space**](https://nftdrive.net/?nav=space)
  * 스페이스는 NFTDrive에서 발행한 NFT를 공유할 수 있는 곳입니다. 사용자는 제공되는 다양한 유형의 자산을 확인하고 입찰할 수 있습니다. NFTDriveEX를 사용하면 복잡한 로그인 등을 할 필요 없이 즉시 공유를 시작할 수 있습니다. 사용자는 주소를 등록하기만 하면 일정 금액을 예치할 필요도 없고, NFT를 전송하기 위한 신청 절차도 필요 없습니다.

이 웹사이트는 Symbol blockchain 의 기본 통화인 XYM 으로 결제할 수 있는 장소의 목록입니다.
결제에 사용되는 암호화 자산은 XYM 입니다. 그러나 점점 더 'Symbol Payments'로 불리고 있습니다.
Symbol 은 결제 전용으로 설계된 블록체인은 아니지만, 송금 속도, 상대적으로 낮은 거래 수수료, XYM 을 주고받는 데 사용되는 다양한 지갑, 송장용 QR코드 생성 애플리케이션 덕분에 Symbol payments 가 인기를 끌 수 있었습니다. 또한, 현재 커뮤니티 자원 봉사자들에 의한 탭 투 페이 디바이스  "CryptoPiTouch" 의 개발이 진행 중이며, 2024년 5월 블록체인 엑스포 스프링에서 "CryptoPiTouch" 프로토타입 디바이스가 공개되어 작동 방식을 시연했으며, 현재 애플리케이션과 하드웨어의 완성을 위해 제작 중입니다. 이 시스템이 완성되면 Symbol payments 행사장에서 시범 운영될 예정입니다.
