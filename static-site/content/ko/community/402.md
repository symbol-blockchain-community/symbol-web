## "My Ashiato" 에 대하여
"My Ashiato" 와 함께 모든 순간을 기억하세요. 이 앱은 좋아하는 장소와 특별한 추억을 텍스트, 사진, 동영상으로 핀으로 표시하여 당신만의 맞춤형 인생 지도를 만들 수 있게 해줍니다. 각 순간의 감정, 느낌, 장면을 포착하고 언제든지 되돌아볼 수 있습니다.
![My Ashiato pic.png](../images/My_Ashiato_pic_9171795ce9.png)


## 주요 기능
### 무엇을 기록할 수 있을까
- 핀 (하루 15회)
- 문자 (하루 5건)
- 사진 (하루 5장)
- 비디오 (하루 5개) 

위치와 시간이 녹음과 자동으로 연동됩니다. 녹음된 데이터는 다음 날부터 Symbol 블록체인에 암호화된 형태로 저장될 수 있습니다.

Symbol 블록체인에 기록된 데이터는 텍스트 데이터이며, 기록 시간과 기록 위치를 포함합니다. 사진과 동영상은 앱 내의 애플리케이션 데이터로 로컬에 저장됩니다.

### 데이터 관리
- 일일 데이터 제한: 992바이트(압축됨)
- 데이터를 Symbol에 기록할 때, 다음 날부터 7일 이내에 수동으로 기록해야 합니다.
- 7일이 지나면 처리되지 않은 데이터는 자동으로 삭제됩니다.

### 계정 정보
- Symbol 계정 정보, 언어 설정(15개 언어 사용 가능) 및 사용 현황을 확인하세요.
- 기존 계정은 가져올 수 없습니다. "My Ashiato" 초기 설정에서 새 계정을 생성해 주세요.
- 계정을 설정할 때, Symbol에 암호화하여 저장한 데이터를 복호화하는 데 사용되는 뷰 코드가 생성됩니다. 뷰 코드가 다른 사람에게 알려지면 기록된 데이터를 볼 수 있으므로 개인 키나 mnemonic 과 동일한 주의로 안전하게 보관해 주시기 바랍니다.

### 제한사항
- 동일한 위치에 기록은 하루에 한 번으로 제한됩니다.
- GPS 정확도로 인해 정렬이 맞지 않을 수 있습니다.
- Symbol 블록체인에 기록하려면 Symbol 블록체인의 기본 통화인 XYM 의 거래 수수료가 필요합니다.
