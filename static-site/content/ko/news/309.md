## 요약
2025년 9월 23일, 공식 심볼 블록체인 X 계정（[@thesymbolchain](https://x.com/thesymbolchain)）은 중대한 [네트워크 업그레이드](https://x.com/thesymbolchain/status/1970323150253781038)를 발표했습니다.

이 업그레이드는 블록 높이 `4,759,100`에서 발생할 예정이며, 이는 9월 25일 23:30(UTC)경으로 추정됩니다.

이번 업그레이드는 Symbol 네트워크 내 잠재적 공격 경로를 해결하며, 노드 운영자들에게 신속한 업데이트 적용을 권장합니다.

## 업그레이드 세부 정보
이 네트워크 업그레이드는 특정 조건 하에서 집계된 트랜잭션 해시가 고유하지 않게 될 수 있는 버그를 수정함으로써 잠재적인 공격 경로를 해결합니다. 
또한 업데이트된 종속성 라이브러리(CPPZMQ 4.11.0, RocksDB 10.5.1 및 OpenSSL 3.5.2)를 통합하여 네트워크의 전반적인 안정성과 보안을 강화합니다.
추가로 Rest Gateway 버전 2.5.1은 집계 트랜잭션 v3 지원을 도입하고 WebSocket 트랜잭션에서 maxFee 속성이 누락되는 문제를 해결합니다.
이러한 업데이트는 BREAKING FORK이므로 노드 운영자는 사전에 준비 작업을 수행해야 합니다.

- [**Catapult Client v.1.0.3.9**](https://github.com/symbol/symbol/releases/tag/client%2Fcatapult%2Fv1.0.3.9)
  - Fixed:(BREAKING FORK) fix aggregate transaction hash bug where the aggregate transaction hashes were not unique in certain circumstances.
- [**Rest Gateway v2.5.1**](https://github.com/symbol/symbol/releases/tag/rest%2Fv2.5.1)
  - Fixed:WebSocket transaction is missing the maxFee property, which was getting dropped.


## 개발자를 위한 정보
JavaScript 및 Python SDK 버전 3.3.0, 레거시 TypeScript/JavaScript SDK 버전 2.0.6, Java SDK 버전 1.0.4에 대한 업데이트를 권장합니다. 이 업데이트는 분산형 애플리케이션(dApp)과의 호환성을 보장하기 위한 것입니다. 사용자는 목표 블록 높이에 도달한 후 애플리케이션을 출시할 것을 권장합니다.

* [**JS SDK V3.3.0**](https://www.npmjs.com/package/symbol-sdk/v/3.3.0)
* [**Python SDK V3.3.0**](https://pypi.org/project/symbol-sdk-python/3.3.0/)
* [**JS SDK V2.0.6**](https://www.npmjs.com/package/symbol-sdk/v/2.0.6)
* [**Java SDK V1.0.4**](https://github.com/symbol/sdk-java/releases/tag/v1.0.4)


또한 신속한 업그레이드를 지원하기 위해 GitHub에 Node 설정 도구인 Symbol-Bootstrap(버전 1.1.12)과 Symbol-Shoestring(버전 0.2.2)의 업데이트된 버전이 출시되었습니다.

* [**symbol-bootstrap V1.1.12**](https://github.com/symbol/symbol-bootstrap)
* [**symbol-shoestring V0.2.2**](https://pypi.org/project/symbol-shoestring/0.2.2/)

## 영향 및 대응책
이번 업그레이드는 심볼 네트워크의 보안을 크게 강화하고 잠재적 공격 경로를 해결합니다.
특정 조건에서 중복되지 않은 집계 거래 해시를 생성할 수 있는 이 버그는 악의적인 공격자가 네트워크를 교란할 수 있게 할 수 있었습니다. 이 문제를 수정함으로써 네트워크의 안정성이 향상됩니다.

플랫폼 안정성의 지속은 기업 사용자의 비즈니스 연속성에 기여할 것입니다. 그러나 업데이트를 소홀히 하는 노드 운영자와 개발자는 네트워크에서 포크되어 배제될 위험이 있습니다.

업그레이드 시점은 블록 높이에 기반하므로 네트워크 진행 상황에 따라 정확한 시점이 달라질 수 있습니다. 최신 정보는 Symbol 블록체인의 공식 X 계정（[@thesymbolchain](https://x.com/thesymbolchain)） 또는 [Symbol Community Web](https://symbol-community.com/ko) 를 확인하시기 바랍니다.
