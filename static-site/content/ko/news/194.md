symbol-sdk v3 가 업데이트되어 릴리스되었다고 보고되었습니다.

이제 symbol-sdk v3 에서 Typescript 지원을 사용할 수 있습니다. 이를 통해 더 많은 개발자가 보다 편리한 개발 환경에서 사용할 수 있습니다.

이전 JavaScript 버전인 symbol-sdk v3 에는 유형 정의가 없었기 때문에 Typescript 프로젝트에 통합하려면 약간의 독창성이 필요했습니다. 개발자는 유형을 수동으로 정의하거나 별도의 typedef 파일을 만들어야 했기 때문에 기존 Typescript 프로젝트에 통합하는 데 장애가 되었습니다.

그러나 symbol-sdk v3 에 유형 정의 파일을 추가하여 이제 typescript 프로젝트에서 원활하게 사용할 수 있습니다. 이를 통해 개발자는 더욱 강력하고 안전한 코드를 작성하고 전체 개발 프로세스를 보다 효과적으로 관리할 수 있습니다. 또한 자동 완성 및 유형 검사 기능을 통해 오류를 조기에 발견하고 버그 발생을 사전에 방지할 수 있습니다.

이 버전은 `symbol-sdk@3.0.11` 에서 실행됩니다. 샘플 스크립트는 아래에 제공됩니다.
