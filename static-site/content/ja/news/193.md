symbol-sdk v3がアップデートされました。

symbol-sdk v3では Typescript がサポートされ、より多くの開発者がより快適な開発環境で利用できるようになりました。

以前の symbol-sdk v3 JavaScript バージョンでは、型定義が存在しなかったため、typescript プロジェクトに組み込む際には工夫が必要でした。開発者は手動で型を定義するか、別途型定義ファイルを作成する必要があり、既存の Typescript プロジェクトへの導入のハードルとなっていました。

しかし、symbol-sdk v3 に型定義ファイルが追加されたことで、typescript プロジェクトでもスムーズにお使い頂けるようになりました。これにより、開発者はより堅牢で安全なコードを書くことができ、開発プロセス全体をより効果的に管理できるようになりました。また、自動補完や型チェックの恩恵を受けることで、エラーを早期に検知し、バグの発生を未然に防ぐことができるようになります。
