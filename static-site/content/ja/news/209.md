Symbol で DAO 活動を続ける **Symbol Japan Forum** は、この度 Symbol Blockchain Community で開発された OSS を集約する Github Organization を作成致しました。また、同様に Javascript のレジストリである npm レジストリに対しても Organization を作成致しました。

今後、コミュニティで開発された Javascript パッケージは、 **@symbol-blockchain-community/${パッケージ名}** というプレフィックスを付与してアクセスすることが出来ます。

**Github Organization: [Symbol Blockchain Community](https://github.com/Symbol-Blockchain-Community)**

今後、 Symbol コミュニティで開発する OSS 開発者はご賛同頂ける方は是非一緒にリソースの管理、追加にご協力下さい。コミュニティで開発されたソフトウェアをより多くの人々に活用して頂けることを目指します。

なお、最初のリリースとしてネイティブアプリケーションへ symbol-sdk をより簡単に組み込めるようにする為の、 symbol-sdk 向け crypto モジュールの代替パッケージをリリース致します。このパッケージは通常 symbol-sdk を expo のマネージド React Native 環境上で使用しようとすると、依存関係の解消難易度が高かった問題に対して、開発者がより簡単に開発を開始し始めることが出来るようにしたパッケージとなります。

詳細はパッケージのレポジトリ、または npm レジストリを参照下さい。

**Github Repository: [expo-symbol-crypto](https://github.com/Symbol-Blockchain-Community/expo-symbol-crypto)**

**npm Registry: [@symbol-blockchain-community/expo-symbol-crypto](https://www.npmjs.com/package/@symbol-blockchain-community/expo-symbol-crypto)**
