服薬記録トレーサビリティアプリ「DrugN」を開発する株式会社HealthCareGateはGoogle Cloudのプログラムである、Google Cloud Web3 Startup Programに「DrugN」が採択されたことを発表しました。同社はWEB3テクノロジーに特化したGoogle Cloudのプログラムを活用することで世界中の革新的なブロックチェーンスタートアップに追いつきたいとしています。

### Google Cloud Web3 Startup Programとは
> Google for Startups クラウド プログラムを通じて、Web3 プロジェクトとスタートアップは、必要なテクノロジー、コミュニティ、リソースを提供する新しい特典にアクセスできます。分散型アプリ、Web3 ツール、サービスなどを構築する際に、インフラストラクチャのイノベーションに集中できます。
[Google Cloud Web3 Startup Program](https://cloud.google.com/startup/web3?hl=ja)
