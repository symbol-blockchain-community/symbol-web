# 概要

Symbol を活用した新しい仕組みのゲームが登場しました。NFT単体で動作するフルオンチェーンゲームです。この技術では対象のゲームを動作させる為に必要なデータが全てブロックチェーン上に格納されており、該当のブロックチェーンが消滅しない限りは存在し続けるゲームとなります。

# 仕組み

何故 NFT 単体でゲームとして格納され、ゲームプレイが可能であるかを紹介します。本フルオンチェーンゲームではまず以下のツールが利用されています。

* ブロックチェーン ... [Symbol](https://docs.symbol.dev/index.html)
* NFT ... [NFT Drive](https://nftdrive.net/)

各ツールの機能と立ち位置を紹介します

## Symbol

Symbol は ブロックチェーン です。実際に NFT としてゲームを稼働させる為のプログラムをNFTとして書き込みます。

## NFT Drive

ゲームの開発者が Symbol に対して簡単に NFT として書き込んだり、ゲームのユーザーがゲームを表示する際に必要な HTML を出力するサービスです。本フルオンチェーンゲームでは Symbol 上に書き込まれている ゲーム のプログラムを NFT Drive が読み込み、適切な HTML を生成、それをユーザーに返す事でユーザーがブラウザで NFT ゲームを楽しむ事ができるようになっています。

もし NFT Drive の仕組み詳細を知りたい場合は、以下の記事を参照下さい

[フルオンチェーン NFT の仕組](https://symbol-community.com/community/8)

# 特徴

フルオンチェーンゲームは以下のような特徴があります。

* プログラムを書き込んだブロックチェーンそのものが消滅しなければ半永久的に消えない
* NFT として書き込めば、システム上はゲームプログラムの所有者を表現する事も可能（法的な判断を要する場合は別途専門家に必ず相談下さい）
* 既存の NFT マーケットプレイス等で販売が可能
