## 2. [株式会社NFTDrive](https://nftdrive.net) (NFTDrive Inc.)
![logo5.png](../images/logo5_87154597be.png)

### プロダクト・サービス

* **NFTDrive**
  * 自動化されたツールでブロックチェーン上に様々な種類（画像・動画・ドキュメント・PDF・HTMLなど）のデジタルデータをアップロードしてフルオンチェーンNFTを発行します。フルオンチェーンNFTとは、一般的なIPFS等によるデジタルデータをブロックチェーン外に保存する方式ではなく、トークンとデジタルデータを紐づけた状態とし、NFTに関する全てのデータをブロックチェーン上に保存している極めて耐改竄性が高く、永続的なNFTです。NFTDriveではデータの耐改竄性を向上するために、世界で始めて秘密鍵を破棄させる方法を採用することで、絶対的に変更不可能なデータ領域を開発しました。（※特許出願中）
* **NFTDriveEX**
  * 全ての人にフルオンチェーンNFTのP2P取引を可能にすることを目的に開発されたNFT特化型の国産ウォレットです。
* **HAYABUSA**
  * フルオンチェーンジェネラティブNFT作成ツールで、ランダムマップデータと画像データを１つにまとめたNFGファイルを開発したことで、ガス代を節約しつつジェネラティブデータは数千枚分のデータをSymbolブロックチェーンへ僅か１回の書き込みで完成します。FreeMintやSaleMintも可能、MintSiteの提供を行っており、Al（ホワイトリスト）対応、Reveal機能を備え、EVMに対応しているチェーンであればほとんどのチェーンに対応することができます。また、フルオンチェーンで作成する必要がない場合、GoogleDriveからのオフチェーンジェネラティブNFTを作成することができます。
* **Space**
  * NFTDriveで発行したNFTを共有することができる空間がSpaceです。様々な種類のアセットを見てオファーすることができます。NFTDriveEXを使用すれば、煩わしいログイン等をすることなくそのまま取引できます。アドレスを登録するだけで、一定の残高を預け入れることなく、NFTの入出庫をする際に、依頼の手続きも不要です。
![NFTDrive.jpg](../images/NFT_Drive_f5314ce747.jpg)

## 4. [有限会社オープンスフィア](https://opensphere.co.jp) (OPENSPHERE Inc.)
![osi_banner.png](../images/osi_banner_bffae08b29.png)

### プロダクト・サービス
* [**QUEST**](https://quest-bc.com)
![quest.jpg](../images/quest_7554f1af9a.jpg)
  * Symbol（XYM）を使った全く新しい投げ銭プラットフォームです。視聴者は、配信者が行う「クエスト」に対してXYMを投げ銭（ブースト）できます。XYMはクエストを期限内に達成するまでプールされ、達成は投票で決まります。達成が承認されれば投げ銭されたXYMは挑戦者へ送金され、達成できなければ、XYMは視聴者に返金されます。この一連の流れは、Symbolブロックチェーンを用いたスマートコントラクトで自動的に執行されます。
    * [ブロックチェーン技術を利用した新しい投げ銭プラットフォーム「QUEST」をMainnetにてローンチしました](https://blog.opensphere.co.jp/posts/release002)![quest_mechanism.png](../images/quest_mechanism_e0e303b36e.png)
* [**Metal on Symbol V2**](https://github.com/OPENSPHERE-Inc/metal-on-symbol)
  * Metal（メタル）とは Symbol ブロックチェーンに任意の（サイズの）データを書き込んだり読み込んだりするためのプロトコルで、Symbol ブロックチェーンをオンラインの不揮発性メモリ（ROM）として使用できます。プロトコル及び CLI と SDK がオープンソースとして公開されています。

---

#### 有限会社オープンスフィアは、ソフトウェアの開発や、保守・運用、サービス運営支援を行うソフトウェア技術会社です。Symbolを用いたプロダクト・サービスとして、上記QUEST、Metal on Symbol V2の他、概念実証(PoC)として[「Smart Transaction for Symbol (PoC)」](https://github.com/OPENSPHERE-Inc/smart-transaction-poc)を開発、公開しています。
