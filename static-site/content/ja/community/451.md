## イベント概要
ブロックチェーンを活用したアプリ開発を学ぶハンズオンワークショップを[NEMTUS](https://nemtus.com)が7月19日に沖縄県宜野湾市で開催します。

![Meetup "Play with Blockchain" @CODE BASE Okinawa.png](../images/Meetup_Play_with_Blockchain_CODE_BASE_Okinawa_64d622ee29.png)

### ブロックチェーンに触れよう会@CODE BASE Okinawa
* 日時 : 2025年7月19日(土)13:00〜17:00
* 会場 : [CODE BASE Okinawa](https://www.protosolution.co.jp/codebase/)
* 住所 : [沖縄県宜野湾市大山7-10-14 プロト宜野湾沖縄本社第2ビル 2F](https://www.google.com/maps/@26.284167,127.744694,998m/data=!3m1!1e3?hl=ja&entry=ttu&g_ep=EgoyMDI1MDYwNC4wIKXMDSoASAFQAw%3D%3D)

* 持ち物 : ノートPC(Mac,Windowsどちらでも）、電源コード
