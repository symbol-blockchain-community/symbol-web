## Quest の紹介

[QUEST](https://quest-bc.com/) とは「成功報酬型」投げ銭プラットフォームです。

このプラットフォームでは、ロールプレイングゲームでクエストを受ける感覚で、あなたのリアルなプロジェクトやタスク、チャレンジをクエストとして実行し、 あなたのを応援するフォロワーから投げ銭を受け取ることができます。 しかし、この投げ銭はクエストが成功したと認められた時、始めて獲得することができる仕組みになっています。

QUEST はこの仕組みを実現する為に、Symbolブロックチェーン を基盤とした暗号資産による決済システムを利用しています。仮想通貨で取引するため、投げ銭を受け取るために銀行口座を用意する必要はありません。

## Quest の2つの機能

Quest には "クエスト" と "オファー" と呼ぶ２つの機能があります。この機能を紹介致します。

### クエスト

クエスト機能ではあなた自身が目標を立て、その進捗や結果を Quest に対して投稿していく事で投げ銭を受け取る事が出来ます。クエストの立て方は公式ホームページの以下リンクを参照しましょう

[ガイド - クエスト作成](https://quest-bc.com/docs/guide/quest-creation)

クエストを立てて進捗を報告していくと、以下画面のように Symbol を使って投げ銭が届き始めます。コストのかかる活動ならこれを活動資金に、そうでなくともこれを励み、もしくは報酬にクエストの実行を完了させていきます

![symbol quest xym 投げ銭](../images/symbol_quest_ui_2c57699530.png)

勿論、クエストの実行者が失敗しタスクが完了しない事もあります。 Quest では投票機能が提供されており投げ銭の実施者はクエストの内容を評価して最終的に支払いを行うか否かを決める事が出来ます。

![symbol quest xym 投げ銭](../images/symbol_quest_vote_941c2fd810.png)

クエストの完了時はこの投票が最終的に行われて、承認されるとクエスト実行者は投げ銭として報酬を受け取る事ができます。

### オファー

オファー機能では誰かに仕事やタスクをお願いする事が出来ます。具体的には仕事やタスクの募集をQuest上で行なって、対応者からの応募が届いたらその方へブーストして報酬のお支払いをします

オファーの立て方は公式ホームページの以下リンクを参照しましょう

[ガイド - オファー](https://quest-bc.com/docs/guide/offer)

オファーを立てると以下のように予算やブースト回数を指定して対応者を募集することが出来ます。

![symbol quest xym 投げ銭](../images/symbol_quest_offer_0b20af6b3d.jpeg)

また、オファーへ応募者が参加してくると以下のように参加者一覧や状態を閲覧出来ます。この中から条件に一致した対応者を選び、予算の範囲内でブーストを行なっていきます。

![symbol quest xym 投げ銭](../images/Fo_K082qa_IAA_Ph_Q5_84343f36bb.jpeg)

ブーストを行う際には以下のように相手をどこまで信頼するか、によって投票時の条件も指定出来ます。これを活用して応募者に対して支払い意思を明示しつつ、タスクに取り組んで貰う事が出来ます

![symbol quest xym 投げ銭](../images/symbol_quest_offer2_22eaddf87f.jpeg)

以上、Quest の "クエスト"と "オファー" 機能の紹介となりました

Quest を活用して新たな価値の交換手段を体験しましょう。また Quest では Boost の実施事に Symbol のオンチェーンで SecretLock Transaction というコントラクトを生成しています。ブロックチェーンの機能を活用する事で Quest の参加者が安心して投げ銭出来る仕組みを用意しています。 ブロックチェーンを用いたコントラクトの構築方法は以下の学習コンテンツも活用下さい

[速習Symbol](https://learn.ja.symbol-community.com/)
