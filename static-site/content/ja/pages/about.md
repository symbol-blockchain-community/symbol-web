## Symbol Community Web チームについて
Symbol Community Web は、Symbol/NEM に関する情報をより見つけやすく、より参加しやすくするために運営されているオープンソースのコミュニティサイトです。新しいユーザーが安心して学べる入口を整えながら、既に活動している人たちの取り組みも継続的に紹介していきます。

## 目標
私たちは次の目標を掲げています。

- 新しいユーザーが安全にブロックチェーンに触れる事が出来る
- 新しいユーザーが Symbol/NEM を使い始めるのを助ける
- Symbol/NEM の最新の情報を届ける
- 既に活動する人々と新しいユーザーを繋ぐ
- コミュニティによって作成されたリソースを紹介する
- ブロックチェーンで社会の課題と向き合うきっかけを作る

## コントリビューションについて
Symbol Community Web は、記事の寄稿、掲載依頼、文言修正、機能改善の提案を歓迎しています。サイトを継続的に更新していくため、現在は GitHub を窓口として運用しています。

### 記事掲載を依頼する場合
記事掲載を依頼する場合は、[Issues](https://github.com/symbol-blockchain-community/symbol-web/issues) に新しい issue を作成してください。issue の本文は、そのまま公開対象として扱われます。掲載を円滑に行うため、次のルールに沿って投稿してください。

1. issue の本文には、掲載したい記事のテキストと画像だけを含めてください。
2. 画像は issue 本文に必ず埋め込んでください。外部リンクの画像は受け付けません。
3. 定期的に AI が issue 本文を確認し、公開処理を行います。確認タイミングは不定期のため、掲載時期に関する問い合わせは受け付けません。
4. AI は issue の本文全体を掲載用テキストとして扱います。挨拶、補足、メモなど、記事以外の余計なテキストは含めないでください。
5. issue の本文の言語は問いません。内容に応じて AI が適切な言語へ変換して掲載します。

### Pull Request で直接反映したい場合
記事本文を直接追加したい方、既存記事やページの文言を修正したい方、サイトそのものを改善したい方は、[Pull Requests](https://github.com/symbol-blockchain-community/symbol-web/pulls) から直接提案してください。内容が明確であれば、記事寄稿でもサイト改善でも歓迎します。

## 参加してくれる方へ
このサイトは、コミュニティの活動や知見を公開し続けることで価値が出ます。ニュース、イベントレポート、ツール紹介、ユースケース、文言修正、小さな改善提案まで、寄稿の形は問いません。公開したい情報や改善したい箇所があれば、まずは issue か pull request で共有してください。
