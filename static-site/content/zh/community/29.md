## NEM/Symbol的Dapps和服务介绍页面。

### **钱包**
#### `Symbol`
* [**Desktop wallet**](https://github.com/symbol/desktop-wallet/releases) Linux, Windows and Mac
* [**Mobile wallet** Android](https://play.google.com/store/apps/details?id=nem.group.symbol.wallet&hl=ja&gl=US) 
* [**Mobile wallet** iOS](https://apps.apple.com/us/app/symbol-wallet/id1528741845) 
* [**Mobile wallet AEM+** Android](https://play.google.com/store/apps/details?id=com.aem.plus) 
* [**Mobile wallet AEM+** iOS](https://apps.apple.com/jp/app/aem/id1607457067) 
AEM+钱包是一款多链(XYM、XEM、BTC、ETH)手机钱包，可让您在任何设备上管理加密交易。设计用于将您所有的加密钱包存储在您手中的一个地方。
* [**Mobile wallet EXYM** Android](https://play.google.com/store/apps/details?id=com.ry0chang.EXYM) 
* [**Mobile wallet EXYM** iOS](https://apps.apple.com/jp/app/exym/id6449445030) 
EXYM 是一款全新风格的加密货币钱包应用，让您轻松在日常生活中享受数字艺术。
它允许用户轻松创建自己的外观设计钱包，打造一种超越传统 NFT 使用的全新体验。对于艺术创作者而言，这提供了一个将自己的艺术作品转化为新价值形式的机会。
* [**Mobile wallet CoinPocket** Android](https://play.google.com/store/apps/details?id=net.teritaris.coinpocket&hl=ja&gl=US)
* [**Mobile wallet CoinPocket** iOS](https://apps.apple.com/jp/app/coinpocket/id6479215775)
CoinPocket 的目标是简单易用，而不像其他加密钱包那样复杂。
* [**Browser wallet NFT-DriveEX**](https://chrome.google.com/webstore/detail/nftdriveex/oodjpaddnaphopckiphmgapkafegcghl)
NFTDriveEX 是一个点对点钱包（P2P)，允许用户在网站的任何地方进行 NFT、XYM 和其他马赛克的交换。
[Google Chrome 扩展版本](https://chromewebstore.google.com/detail/nftdriveex/oodjpaddnaphopckiphmgapkafegcghl?hl=ja)
[Web（通用版本）](https://nftdrive-ex.net)
* [**Browser wallet SSS(Safely Sign Symbol) Extension**](https://chrome.google.com/webstore/detail/sss-extension/llildiojemakefgnhhkmiiffonembcan)
Symbol 区块链 SSS Extension 是一个 Google Chrome 扩充功能，与使用 Symbol 区块链的 Web 应用程式相容。这个 SSS 应用程式是为了在 Symbol 区块链上签署交易而设计的，而不需要处理私钥。虽然这个应用程式是一个允许您安全签署交易的扩充功能，但也可以用作浏览器钱包。
* [**Browser wallet Ventus Wallet**](https://ventus-wallet.tk)
这是一个使用Google Chrome扩充功能“SSS Extension”的Symbol网页钱包。
* [**Browser wallet One Plate**](https://oneplate.food-nft.io)
ONEPLATE 是一个简单易用的 NFT 开始工具。
它的设计旨在易于理解和使用，目标是实现“现在任何人都能够做到”的目标。

* [**QUEST**](https://quest-bc.com)
QUEST是一个建立在区块链技术上的新型“律师费用基础”打赏平台。 "Quest"是你在挑战中追求的成就。
支持者期待着你的奋斗之旅和成功的结局，通过捐款来增加"Quest"的金库。
当你完成"Quest"时，你将获得金库内容。如果你没有完成，金库内容将退还给捐款者。
但是，是谁判断"Quest"是成功还是失败呢？是支持者的投票。
QUEST的主要特点是全自动和安全的处理，实现了区块链技术。

TokenLive 是一个利用 Symbol 区块链 Mosaic(Token) 的直播串流服务。只有拥有分配者指定的特定 NFT (唯一的) 或不可转让的 FT(bundle) 的人才能参加。

### **NFT**
#### `Symbol`

* [**NFT-Drive**](https://nftdrive.net)
NFT-Drive是一个数字内容上传器，允许您创建数字资产，例如非同质化代币（NFT）和可转换代币（FT）。"Fully-on-chain NFT".
-图像档案：GIF、JPEG、PNG、SVG
-音讯档案：MP4、MP3、MIDI
-影片档案：MP4、AVI
-其他档案：TXT、PDF、HTML（CSS、JavaScript）、ZIP
* [**Heartlog**](https://heartlog.net)
Heartlog是一个将每一个重要的记忆和无可替代的时刻储存为完全在链NFT的服务。包括“宝宝命名书法艺术NFT”、“纪念NFT”、“婚礼NFT”、“讯息NFT”、“宠物NFT”等，将您的数位档案化为完全在链NFT，并使用NFT-Drive储存到区块链上。无法被删除、防篡改且永久保存。
支援的档案格式包括：
-图片档案：GIF、JPEG、PNG、SVG；
-音讯档案：MP4、MP3、MIDI；
-影片档案：MP4、AVI；
-其他档案：TXT、PDF、HTML(CSS Javascript)、ZIP。
* [**UMUI**](https://umui.okinawa/)
UMUI 是一项服务，它以照片、视频和音频文件的形式将您在冲绳旅行、重要纪念日、婚礼以及在冲绳度过的珍贵家庭时光存储在Symbol公共区块链上。
-图片档案：GIF、JPEG、PNG、SVG；
-音讯档案：MP4、MP3、MIDI；
-影片档案：MP4、AVI；
-其他档案：TXT、PDF、HTML(CSS Javascript)、ZIP。
* [**GRAPEs VirtualCardToken Generator**](https://nftdrive-explorer.info/download.php?address=21FA41D6400ADB00)
“GRAPEs VirtualCardToken Generator”是一个WebAPP服务，它可以生成文字、图像等内容并输出为一个HTML档案。生成的档案可以作为单一FT/NFT上链，并以对应的全上链FT/NFT储存在区块链上。
