## NFTDriveEX 正式推出全面兼容 Ethereum Virtual Machine (EVM) 及其他多个区块链的全新版本，赋予用户更广泛的数位资产管理权能。

[NFTDrive Inc.,](https://nftdrive.net/) 总部位于日本京都，由首席执行官中岛理男领导，欣然宣布推出最新力作 - 日本NFT专用钱包[NFTDriveEX](https://nftdrive.net/?nav=ex)，这款创新钱包将全面支援多个区块链，包括 Ethereum（EVM）和第二层解决方案，为用户提供更广泛、更便利的数位资产管理体验。

![NFTDriveEX_2.png](../images/NFT_Drive_EX_2_a2882a24b6.png)


NFTDriveEX，拥有直觉简约的使用者界面，广受欢迎。

最新的功能旨在进一步提升NFTDriveEX的便利性。这项多链技术允许加密资产和NFT在不同区块链之间自由转移，弥补了潜在的区块链问题，包括成本、规模和发行量。

目前，众多NFT数据存在于区块链之外，面临丢失风险。 （※１）

Symbol区块链不需要EVM（Ethereum Virtual Machine ※2）即可使用，避免了智能合约可能的意外行为。在Symbol中，NFTs可轻松铸造成完全的链上NFTs，成为链上数据存储的一部分。据称，这提高了传统NFT的可靠性。

NFTDrive目前采用一种"混合全链上"技术，主要利用存储在Symbol上的全链上数据作为主数据，同时作为其他链上NFT的参考。这使得这些NFT可以在主要NFT市场上轻松上市。同时，该技术允许在Symbol之外的其他链上进行铸造和拥有的NFT轻松切换到Symbol，实现区块链间更自由、更开放的P2P无信任交易，无需透过第三方市场。

在支付方面，NFTDrive支持以太坊（第二层）上的ETH、多种ERC20代币和稳定币。这意味着可以灵活地在不同区块链上铸造NFTDrive的全链上NFT，同时在以太坊和其他链上查看和发送NFT等功能也能毫不费力地实现，无需进行修改。

（※１）这是因为许多NFT在技术上仅在区块链上存储服务器的参考URL，该服务器存储与NFT相关的“链外”数字数据（如图像）。

（※２）在区块链上执行智能合约的程序主体。它只执行源代码，因此无法确定服务的结果是否正确。因此，在使用源代码时进行审计是必要的。
