创意区块链解决方案的先行者 [Opening Line公司](https://www.opening-line.co.jp) 宣布即将出版 "使用Symbol区块链构建的实际应用" 一书。

![OpeningLine_book.png](../images/Opening_Line_book_50cab245ec.png)

这是一本实用指南，详细介绍了如何利用 Symbol 区块链的技术优势，使用 TypeScript 和 Python 开发应用程序。
这本书对 Symbol 开发人员和有其他区块链经验或对 blockchain 服务感兴趣的人都很有用。本书涵盖了各种领域，如 DAO 演示应用程序的示例和设计背景，以及跨链集成。
