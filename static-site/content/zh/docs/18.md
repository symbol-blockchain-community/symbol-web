## 什么是 Symbol 的钱包？
Symbol 的钱包是一个允许您操纵您的 Symbol 帐户的应用程序。 它使您可以访问自己手中的资产。 您需要一个钱包来转移资金和管理您的 XYM。 使用钱包，您可以查看余额、发送交易和参与收获。
钱包只是管理您的 Symbol 账户的工具。 钱包可以随时兑换。 大多数钱包还允许您从单个应用程序管理多个 Symbol 帐户。
那是因为你的资产不在钱包里，而是你自己通过钱包在区块链上管理你的资产。 钱包只是一种管理资产的工具。

＃＃ 资产管理
从钱包中您可以查看您的账户余额、交易历史、收获历史以及发送和接收资产。 一些钱包可能具有发行自己的代币或展示 NFT 的能力。

[more on Mosaic](https://docs.symbol.dev/concepts/mosaic.html)

Symbol 功能允许您发行自己的唯一令牌（马赛克）。 马赛克可以设置为不可转移、可转移或可检索。
使用示例包括积分和份额共享、签名、投票和其他资产。

### 马赛克细节
#### 参与委托收获
任何拥有 10000 个或更多 XYM 的人都可以参与收获（质押）。 通过委托给运营的节点，可以获得一定概率的收获费。 收获的概率随重要性而变化，与节点重要性或委托数量无关。

### 使用多重签名进行账户管理
Multi-sig 是一种通过在帐户中移动资产时使用多个私钥签名来提高安全性的功能。 虽然普通帐户允许您使用单个私钥移动资产，但多重签名允许您保护您的资产，即使您的私钥之一被其他人知道或者您通过使用多重签名联合签名丢失了您的私钥。 一些钱包允许您设置此多重签名。

## 安装钱包
钱包主要有两种类型：桌面钱包和移动钱包。 移动版推出了多个社区版本，并提供了独特的功能。
### 桌面钱包
![symbol_wallet_shot_image.png](../images/symbol_wallet_shot_image_62c2224d25.png)

您可以下载适用于 windows/mac/linux 的相应钱包。 在执行维护时始终安装最新版本。

#### 步骤１
从下面的链接下载最新版本的资产安装程序。
[Link for desktop wallets](https://github.com/symbol/desktop-wallet/releases)

* windows
  * symbol-desktop-wallet-win-x.x.xx.exe
* mac
  * symbol-desktop-wallet-mac-x.x.xx.dmg
* linux
  * symbol-desktop-wallet-linux-amd64-x.x.xx.deb
  * symbol-desktop-wallet-linux-amd64-x.x.xx.snap
  * symbol-desktop-wallet-linux-x64-x.x.xx.tar.xz

#### 步骤２
运行安装程序并按照安装说明进行操作。
#### 步骤３
##### 创建个人资料
请记住备份在创建配置文件时创建的助记词，因为它很重要。 单击下面的链接以查看创建配置文件的指南（创建帐户）。
[Creating an account by docs.symbol.dev](https://docs.symbol.dev/guides/account/creating-an-account.html)

助记词是在上面的网站上复制的，但是如果你点击同屏的下载按钮，你可以下载一个包含[**地址**，**助记词**，**每个二维码**的PDF文件] ]. 将此文件保存在其他人无法找到的安全位置。
