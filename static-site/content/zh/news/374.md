# 2026年2月博客（日文译）

那个……稍微……让我喘口气……再坚持一下下…… 😮‍💨

大家好！2月真的非常艰难。我几乎没有休息地反复写作和审阅，面向开发者的教程数量也创下了历史新高！

但还没有结束。也就是说，在我……没错，回到那座我最喜欢的城市之前，这种节奏恐怕还要持续一阵子。

因此，这次报告相当朴素。只有图表，没有截图。我知道，你们已经在嘘我了。

## 本月工作概要

- **内容撰写**
**开发者页面**（task-docs-dev）：我们制作了关于多签账户[创建](https://docs.symboltest.net/en/devbook/accounts/configure-multisig/)与[签名](https://docs.symboltest.net/en/devbook/accounts/sign-multisig/)、[交易赞助（代付 Gas）](https://docs.symboltest.net/en/devbook/transactions/fee-sponsorship/)、以及[账户限制新增](https://docs.symboltest.net/en/devbook/accounts/account-restrictions/)的教程。

  **开发者页面**（task-docs-dev）：我们在参考指南中新增了[序列化表](https://docs.symboltest.net/en/devbook/reference/serialization/)和[WebSocket API](https://docs.symboltest.net/en/devbook/reference/websockets/)。

  **教材页面**（task-docs-textbook）：我们在教材中新增了[元数据](https://docs.symboltest.net/en/textbook/metadata/)页面。
- **审阅**
**开发者页面**（task-docs-dev）：
[Zero](https://github.com/zero4862) 制作了新教程，内容包括将命名空间链接到[地址](https://docs.symboltest.net/en/devbook/namespaces/link-namespace-to-address/)与[马赛克](https://docs.symboltest.net/en/devbook/namespaces/link-namespace-to-mosaic/)，为[账户](https://docs.symboltest.net/en/devbook/namespaces/namespace-metadata/)、[命名空间](https://docs.symboltest.net/en/devbook/namespaces/namespace-metadata/)和[马赛克](https://docs.symboltest.net/en/devbook/mosaics/mosaic-metadata/)添加元数据，以及[新建马赛克](https://docs.symboltest.net/en/devbook/mosaics/create-mosaic/)、[修改马赛克定义](https://docs.symboltest.net/en/devbook/mosaics/modify-mosaic-definition/)、[命名空间回执](https://docs.symboltest.net/en/devbook/namespaces/resolve-namespace-from-receipt/)、[马赛克回收](https://docs.symboltest.net/en/devbook/mosaics/revoke-mosaic/)和[链状态](https://docs.symboltest.net/en/devbook/chain/chain-heights/)等内容。以上均已审阅并发布。

  **NEM**：[CryptoBeliever](https://github.com/cryptoBeliever) 完成了非常出色的工作：为 [NIS1 REST API](https://github.com/NemProject/nem/tree/dev/openapi) 编写了完整的 [OpenAPI](https://www.openapis.org/) 规范。总计 180 个文件、超过 4,500 行 YAML，审阅沟通持续了数周。这些内容将用于即将发布的全新 NEM 文档。

  现在只差说服他也为 Symbol 的 REST API 做同样的事了。我认为[当前文档](https://docs.symboltest.net/en/devbook/reference/rest/symbol/)还有很大提升空间 😊
- **管理**
我为新 NEM 文档网站的开发制定了路线图，并接受了艺术团队的视觉指导。

  由于接下来一个月有两位作者将全职投入教程撰写，我们需要时间准备任务清单和作业批处理。

总之，2月就是这样：工作、工作、还是工作，完全没有娱乐。也就是没有 XYM City。不过这其实对游戏反而是好事！拉开一点距离后，我能以更清新的状态重新审视它。等我回到 XYM City 时，会带着爆发性的能量，以及更多可聊的内容回来。

## 接下来计划

Zero 和我会继续推进 Symbol 文档冲刺，直到教程全部完成。

之后，他会转去新 NEM 文档，我会回到审阅工作……然后再投入 XYM City。

另外，新版 Symbol 钱包也在顺利推进，完成后我会立刻写指南。与开发者教程相比，这些会更容易一些。

我还没有完全脱身，但已经看见隧道尽头的光了！而且还是霓虹色的。😎

感谢你读到这里。我们下个月再见！
