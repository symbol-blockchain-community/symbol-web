## 什麼是 Symbol 的錢包？
Symbol 的錢包是一個允許您操縱您的 Symbol 帳戶的應用程序。 它使您可以訪問自己手中的資產。 您需要一個錢包來轉移資金和管理您的 XYM。 使用錢包，您可以查看餘額、發送交易和參與收穫。
錢包只是管理您的 Symbol 賬戶的工具。 錢包可以隨時兌換。 大多數錢包還允許您從單個應用程序管理多個 Symbol 賬戶。
那是因為你的資產不在錢包裡，而是你自己通過錢包在區塊鏈上管理你的資產。 錢包只是一種管理資產的工具。

## 資產管理
從錢包中您可以查看您的賬戶餘額、交易歷史、收穫歷史以及發送和接收資產。 一些錢包可能具有發行自己的代幣或展示 NFT 的能力。

[more on Mosaic](https://docs.symbol.dev/concepts/mosaic.html)

Symbol 功能允許您發行自己的唯一令牌（馬賽克）。 馬賽克可以設置為不可轉移、可轉移或可檢索。
使用示例包括積分和份額共享、簽名、投票和其他資產。

### 馬賽克細節
#### 參與委託收穫
任何擁有 10000 個或更多 XYM 的人都可以參與收穫（質押）。 通過委託給運營的節點，可以獲得一定概率的收穫費。 收穫的概率隨重要性而變化，與節點重要性或委託數量無關。

### 使用多重簽名進行賬戶管理
Multi-sig 是一種通過在帳戶中移動資產時使用多個私鑰簽名來提高安全性的功能。 雖然普通帳戶允許您使用單個私鑰移動資產，但多重簽名允許您保護您的資產，即使您的私鑰之一被其他人知道或者您通過使用多重簽名聯合簽名丟失了您的私鑰。 一些錢包允許您設置此多重簽名。

## 安裝錢包
錢包主要有兩種類型：桌面錢包和移動錢包。 移動版推出了多個社區版本，並提供了獨特的功能。
### 桌面錢包
![symbol_wallet_shot_image.png](../images/symbol_wallet_shot_image_62c2224d25.png)

您可以下載適用於 windows/mac/linux 的相應錢包。 在執行維護時始終安裝最新版本。

#### 步驟１
從下面的鏈接下載最新版本的資產安裝程序。
[Link for desktop wallets](https://github.com/symbol/desktop-wallet/releases)

* windows
  * symbol-desktop-wallet-win-x.x.xx.exe
* mac
  * symbol-desktop-wallet-mac-x.x.xx.dmg
* linux
  * symbol-desktop-wallet-linux-amd64-x.x.xx.deb
  * symbol-desktop-wallet-linux-amd64-x.x.xx.snap
  * symbol-desktop-wallet-linux-x64-x.x.xx.tar.xz

#### 步驟２
運行安裝程序並按照安裝說明進行操作。
#### 步驟３
##### 創建個人資料
請記住備份在創建配置文件時創建的助記詞，因為它很重要。 單擊下面的鏈接以查看創建配置文件的指南（創建帳戶）。
[Creating an account by docs.symbol.dev](https://docs.symbol.dev/guides/account/creating-an-account.html)

助記詞是在上面的網站上複製的，但是如果你點擊同屏的下載按鈕，你可以下載一個包含[**地址**，**助記詞**，**每個二維碼**的PDF文件] ]. 將此文件保存在其他人無法找到的安全位置。
