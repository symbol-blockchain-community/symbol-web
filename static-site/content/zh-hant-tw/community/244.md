## NFTDriveEX 正式推出全面兼容 Ethereum Virtual Machine (EVM) 及其他多個區塊鏈的全新版本，賦予用戶更廣泛的數位資產管理權能。

[NFTDrive Inc.,](https://nftdrive.net/) 總部位於日本京都，由首席執行官中島理男領導，欣然宣布推出最新力作 - 日本NFT專用錢包[NFTDriveEX](https://nftdrive.net/?nav=ex)，這款創新錢包將全面支援多個區塊鏈，包括 Ethereum（EVM）和第二層解決方案，為用戶提供更廣泛、更便利的數位資產管理體驗。

![NFTDriveEX_2.png](../images/NFT_Drive_EX_2_a2882a24b6.png)

NFTDriveEX，擁有直覺簡約的使用者界面，廣受歡迎。

最新的功能旨在進一步提升NFTDriveEX的便利性。這項多鏈技術允許加密資產和NFT在不同區塊鏈之間自由轉移，彌補了潛在的區塊鏈問題，包括成本、規模和發行量。

目前，眾多NFT數據存在於區塊鏈之外，面臨丟失風險。 （※１）

Symbol區塊鏈不需要EVM（Ethereum Virtual Machine ※2）即可使用，避免了智能合約可能的意外行為。在Symbol中，NFTs可輕鬆鑄造成完全的鏈上NFTs，成為鏈上數據存儲的一部分。據稱，這提高了傳統NFT的可靠性。

NFTDrive目前采用一種"混合全鏈上"技術，主要利用存儲在Symbol上的全鏈上數據作為主數據，同時作為其他鏈上NFT的參考。這使得這些NFT可以在主要NFT市場上輕鬆上市。同時，該技術允許在Symbol之外的其他鏈上進行鑄造和擁有的NFT輕鬆切換到Symbol，實現區塊鏈間更自由、更開放的P2P無信任交易，無需透過第三方市場。

在支付方面，NFTDrive支持以太坊（第二層）上的ETH、多種ERC20代幣和穩定幣。這意味著可以靈活地在不同區塊鏈上鑄造NFTDrive的全鏈上NFT，同時在以太坊和其他鏈上查看和發送NFT等功能也能毫不費力地實現，無需進行修改。

（※１）這是因為許多NFT在技術上僅在區塊鏈上存儲服務器的參考URL，該服務器存儲與NFT相關的“鏈外”數字數據（如圖像）。

（※２）在區塊鏈上執行智能合約的程序主體。它只執行源代碼，因此無法確定服務的結果是否正確。因此，在使用源代碼時進行審計是必要的。
