## NEM/Symbol的Dapps和服務介紹頁面。

### **錢包**
#### `Symbol`
* [**Desktop wallet**](https://github.com/symbol/desktop-wallet/releases) Linux, Windows and Mac
* [**Mobile wallet** Android](https://play.google.com/store/apps/details?id=nem.group.symbol.wallet&hl=ja&gl=US) 
* [**Mobile wallet** iOS](https://apps.apple.com/us/app/symbol-wallet/id1528741845) 
* [**Mobile wallet AEM+** Android](https://play.google.com/store/apps/details?id=com.aem.plus) 
* [**Mobile wallet AEM+** iOS](https://apps.apple.com/jp/app/aem/id1607457067) 
AEM+錢包是一款多鏈(XYM、XEM、BTC、ETH)手機錢包，可讓您在任何設備上管理加密交易。設計用於將您所有的加密錢包存儲在您手中的一個地方。
* [**Mobile wallet EXYM** Android](https://play.google.com/store/apps/details?id=com.ry0chang.EXYM) 
* [**Mobile wallet EXYM** iOS](https://apps.apple.com/jp/app/exym/id6449445030) 
EXYM 是一款全新風格的加密貨幣錢包應用，讓您輕鬆在日常生活中享受數位藝術。
它允許用戶輕鬆創建自己的外觀設計錢包，打造一種超越傳統 NFT 使用的全新體驗。對於藝術創作者而言，這提供了一個將自己的藝術作品轉化為新價值形式的機會。
* [**Mobile wallet CoinPocket** Android](https://play.google.com/store/apps/details?id=net.teritaris.coinpocket&hl=ja&gl=US)
* [**Mobile wallet CoinPocket** iOS](https://apps.apple.com/jp/app/coinpocket/id6479215775)
CoinPocket 的目標是簡單易用，而不像其他加密錢包那麼複雜。
* [**Browser wallet NFT-DriveEX**](https://chrome.google.com/webstore/detail/nftdriveex/oodjpaddnaphopckiphmgapkafegcghl)
NFTDriveEX 是一個點對點錢包（P2P)，允許用戶在網站的任何地方進行 NFT、XYM 和其他馬賽克的交換。
[Google Chrome 擴展版本](https://chromewebstore.google.com/detail/nftdriveex/oodjpaddnaphopckiphmgapkafegcghl?hl=ja)
[Web（通用版本）](https://nftdrive-ex.net)
* [**Browser wallet SSS(Safely Sign Symbol) Extension**](https://chrome.google.com/webstore/detail/sss-extension/llildiojemakefgnhhkmiiffonembcan)
Symbol 區塊鏈 SSS Extension 是一個 Google Chrome 擴充功能，與使用 Symbol 區塊鏈的 Web 應用程式相容。這個 SSS 應用程式是為了在 Symbol 區塊鏈上簽署交易而設計的，而不需要處理私鑰。雖然這個應用程式是一個允許您安全簽署交易的擴充功能，但也可以用作瀏覽器錢包。
* [**Browser wallet Ventus Wallet**](https://ventus-wallet.tk)
這是一個使用Google Chrome擴充功能「SSS Extension」的Symbol網頁錢包。
* [**Browser wallet One Plate**](https://oneplate.food-nft.io)
ONEPLATE 是一個簡單易用的 NFT 開始工具。
它的設計旨在易於理解和使用，目標是實現“現在任何人都能夠做到”的目標。

AEM+錢包是一個支援多種加密貨幣（XYM、XEM、BTC、ETH）的行動錢包，讓您可以在任何設備上管理您的加密貨幣交易。它旨在將您所有的加密貨幣錢包集中儲存於手掌中的一個應用程式中。

### **NFT**
#### `Symbol`

* [**NFT-Drive**](https://nftdrive.net)
NFT-Drive是一個數字內容上傳器，允許您創建數字資產，例如非同質化代幣（NFT）和可轉換代幣（FT）。"Fully-on-chain NFT".

-音訊檔案：MP4、MP3、MIDI
-影片檔案：MP4、AVI
-其他檔案：TXT、PDF、HTML（CSS、JavaScript）、ZIP
* [**Heartlog**](https://heartlog.net)
Heartlog是一個將每一個重要的記憶和無可替代的時刻儲存為完全在鏈NFT的服務。包括「寶寶命名書法藝術NFT」、「紀念NFT」、「婚禮NFT」、「訊息NFT」、「寵物NFT」等，將您的數位檔案化為完全在鏈NFT，並使用NFT-Drive儲存到區塊鏈上。無法被刪除、防篡改且永久保存。
支援的檔案格式包括：
-圖片檔案：GIF、JPEG、PNG、SVG；
-音訊檔案：MP4、MP3、MIDI；
-影片檔案：MP4、AVI；
-其他檔案：TXT、PDF、HTML(CSS Javascript)、ZIP。
* [**UMUI**](https://umui.okinawa/)
UMUI 是一項服務，它以照片、視頻和音頻文件的形式將您在沖繩旅行、重要紀念日、婚禮以及在沖繩度過的珍貴家庭時光存儲在Symbol公共區塊鏈上。
-圖片檔案：GIF、JPEG、PNG、SVG；
-音訊檔案：MP4、MP3、MIDI；
-影片檔案：MP4、AVI；
-其他檔案：TXT、PDF、HTML(CSS Javascript)、ZIP。
* [**GRAPEs VirtualCardToken Generator**](https://nftdrive-explorer.info/download.php?address=21FA41D6400ADB00)
「GRAPEs VirtualCardToken Generator」是一個WebAPP服務，它可以生成文字、圖像等內容並輸出為一個HTML檔案。生成的檔案可以作為單一FT/NFT上鏈，並以對應的全上鏈FT/NFT儲存在區塊鏈上。
