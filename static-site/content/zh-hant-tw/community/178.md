HealthCareGate Inc. 宣布他們開發的藥物記錄追蹤應用程式「DrugN」已獲選為 Google Cloud 計畫中的 Google Cloud Web3 Startup Program。該公司希望透過利用谷歌雲在 WEB3 技術領域的專業，來趕上來自世界各地的創新區塊鏈新創公司。

### 什麼是 Google Cloud Web3 Startup Program？
> 透過 Google for Startups Cloud 計劃，Web3 專案和新創公司可以獲得新的優勢，提供所需的技術、社群和資源。現在，在建立去中心化應用程式、Web3 工具、服務等時，您可以專注於基礎架構的創新。
