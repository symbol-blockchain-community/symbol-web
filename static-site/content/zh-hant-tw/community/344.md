## 2. [NFTDrive Inc.](https://nftdrive.net)
![logo5.png](../images/logo5_87154597be.png)

### 產品與服務

* [**NFTDrive**](https://nftdrive.net/?nav=nftdrive)
  * 一種自動化工具，可將各種類型的數位資料（影像、視訊、文件、PDF、HTML 等）上傳至區塊鏈，並發行完全在鏈上的 NFT。 完全上鏈的 NFT 是高度防篡改和持久性的 NFT，它將所有與 NFT 相關的資料儲存在區塊鏈上，代幣和數位資料綁在一起，而不是使用傳統的 IPFS 或其他方法將數位資料儲存在區塊鏈之外。 為了提高資料防篡改能力，NFTDrive 採用全球首創的私密金鑰消減方法，開發出完全不可篡改的資料區域。（*PATPEND）
* [**NFTDriveEX**](https://nftdrive.net/?nav=ex)
  * 日本製造的 NFT 專用錢包，開發目的是讓所有使用者都能以 P2P 方式傳輸完全上鏈的 NFT。
* [**HAYABUSA**](https://nftdrive.net/?nav=hayabusa)
  * 完全在鏈上生成的 NFT 創建工具。 單一 NFG 檔案方法的開發，可將隨機地圖資料與影像資料結合為單一 NFG 檔案，僅需寫入 Symbol 區塊鏈一次，即可完成數千筆產生式資料，同時節省瓦斯費。 FreeMint 或 SaleMint 也是可行的，MintSite 具備 Allow List 支援和 Reveal 功能，可用於大多數支援 EVM 的連鎖店。 如果使用者不需要建立完全上鏈的 NFT，也可以從 GoogleDrive 內建立離鏈產生式 NFT。
* [**Space**](https://nftdrive.net/?nav=space)
  * The Space 是可以共用 NFTDrive 發出的 NFT 的地方。 使用者可檢視並競投所提供的不同類型資產。 有了 NFTDriveEX，使用者可以立即開始分享，無需複雜的登入等程序。 用戶只需登記地址，無需存入一定金額，也無需申請轉帳 NFT 的程序。
