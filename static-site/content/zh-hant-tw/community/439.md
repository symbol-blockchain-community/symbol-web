[Opening Line, Inc](https://www.opening-line.co.jp)創意區塊鏈解決方案的先驅者 [Opening Line公司](https://www.opening-line.co.jp) 宣佈即將推出" 以 Symbol 區塊鏈建立的實用應用" 一書。

![OpeningLine_book.png](../images/Opening_Line_book_50cab245ec.png)

這是一本實用指南，詳細介紹了如何利用 Symbol 區塊鏈的技術優勢，使用 TypeScript 和 Python 進行應用程式開發。
它對 Symbol 開發人員和有其他區塊鏈經驗或對 Web3 服務有興趣的人都很有用。本書涵蓋各種空間，例如 DAO 示範應用程式的範例和設計情境，以及跨鏈整合。
