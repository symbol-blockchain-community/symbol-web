* Quick learning Symbol blockchain JavaScript version / C# version
* Monthly NEMTUS Lightning Talk material collection vol.1
* Monthly NEMTUS Lightning Talk material collection vol.2

### 什麼是技術書籍同人誌博覽會？
> 技術圖書同人誌博覽會（Technical Book Doujinshi Expo）是技術圖書（同人誌）的現場銷售活動。除信息技術外，它還涵蓋科學和工程、數學、設計和管理等廣泛的技術。這項活動是出於支持和加强工程師產出的願望而誕生的。我們的目標是成為一個讓初學者和老手都能放鬆身心、加深知識、結交朋友、在輕鬆互動中成長的場所。
