# Symbol Node RESTFul API 擴展軟件簡介

* [symbol-rest-wrapper](https://github.com/symbol-blockchain-community/symbol-rest-wrapper) 

## 概述
為擴展 Symbol Blockchain Node 的RESTFul API，我們開發了一款新軟件。該軟件與現有的 REST API 配合使用，可執行應用程序開發人員通常要提前進行的繁瑣數據轉換。這樣，開發人員就可以更快、更高效地構建應用程序。它旨在使 Symbol 平台上的應用開發實現無縫對接，並計劃根據開發者的需求擴展其功能。

## 主要特征
- **從地址獲取已處理賬戶信息**: 根據地址提供已處理的賬戶信息。
- **發放臨時秘鑰**: 生成臨時秘鑰。
- **將公鑰轉換為地址**: 根據公鑰生成相應地址。
- **將 hex 格式地址轉換為 1base32 格式**: 支持不同格式之間的地址轉換。
- **將交易 payload 轉換為交易 json**: 將交易 payload 轉換為 JSON 格式。
- **返回實時 nodes 列表**: 提供當前活動 nodes 的列表。
- **創建並返回地址 QR 代碼**: 生成並提供與地址相對應的 QR 碼。

## 擴展使用形象
這種擴展 API 可以在節點內部構建和使用，通過在節點上添加 API 服務器，它可以提供服務，而無需依賴特定的個人開發人員。該應用可作為 Docker 容器或 Node.js 使用，該軟件安裝在現有的 Symbol Node 上，因此額外的環境設置很少。

例如，應用程序開發人員可以將其嵌入到 Google Apps Script 等特殊環境中，而無需使用 Symbol SDK。通過該應用程序接口，他們可以輕鬆實現與 Symbol 相關的操作，減少在瀏覽器環境中設置 Symbol SDK 環境設置的需要，還能降低編寫複雜代碼的頻率。
