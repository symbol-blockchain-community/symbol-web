# 2026年2月部落格（日文譯）

那個……稍微……讓我喘口氣……再一下下就好…… 😮‍💨

大家好！2月真的非常辛苦。我幾乎沒休息地反覆撰寫與審稿，面向開發者的教學篇數也創下歷史新高！

但還沒結束。也就是說，在我……沒錯，回到那座我最喜歡的城市之前，這樣的節奏恐怕還要再持續一陣子。

因此，這次報告相當樸素。只有圖表，沒有截圖。我懂，我已經聽到大家在噓了。

## 本月工作概要

- **內容撰寫**
**開發者頁面**（task-docs-dev）：我們製作了關於多簽帳戶[建立](https://docs.symboltest.net/en/devbook/accounts/configure-multisig/)與[簽名](https://docs.symboltest.net/en/devbook/accounts/sign-multisig/)、[交易贊助（代付 Gas）](https://docs.symboltest.net/en/devbook/transactions/fee-sponsorship/)、以及[新增帳戶限制](https://docs.symboltest.net/en/devbook/accounts/account-restrictions/)的教學。

  **開發者頁面**（task-docs-dev）：我們在參考指南中新增了[序列化表](https://docs.symboltest.net/en/devbook/reference/serialization/)與[WebSocket API](https://docs.symboltest.net/en/devbook/reference/websockets/)。

  **教材頁面**（task-docs-textbook）：我們在教材中新增了[中繼資料](https://docs.symboltest.net/en/textbook/metadata/)頁面。
- **審閱**
**開發者頁面**（task-docs-dev）：
[Zero](https://github.com/zero4862) 製作了新教學，內容包括將命名空間連結到[地址](https://docs.symboltest.net/en/devbook/namespaces/link-namespace-to-address/)與[馬賽克](https://docs.symboltest.net/en/devbook/namespaces/link-namespace-to-mosaic/)，為[帳戶](https://docs.symboltest.net/en/devbook/namespaces/namespace-metadata/)、[命名空間](https://docs.symboltest.net/en/devbook/namespaces/namespace-metadata/)與[馬賽克](https://docs.symboltest.net/en/devbook/mosaics/mosaic-metadata/)新增中繼資料，以及[建立馬賽克](https://docs.symboltest.net/en/devbook/mosaics/create-mosaic/)、[修改馬賽克定義](https://docs.symboltest.net/en/devbook/mosaics/modify-mosaic-definition/)、[命名空間收據](https://docs.symboltest.net/en/devbook/namespaces/resolve-namespace-from-receipt/)、[馬賽克撤回](https://docs.symboltest.net/en/devbook/mosaics/revoke-mosaic/)與[鏈狀態](https://docs.symboltest.net/en/devbook/chain/chain-heights/)等內容。以上皆已審閱並發布。

  **NEM**：[CryptoBeliever](https://github.com/cryptoBeliever) 做了非常出色的工作：為 [NIS1 REST API](https://github.com/NemProject/nem/tree/dev/openapi) 編寫了完整的 [OpenAPI](https://www.openapis.org/) 規格。總計 180 個檔案、超過 4,500 行 YAML，審閱往返持續了數週。這些內容將用於即將發布的全新 NEM 文件。

  現在只差說服他也為 Symbol 的 REST API 做同樣的事。我認為[目前文件](https://docs.symboltest.net/en/devbook/reference/rest/symbol/)還有很大提升空間 😊
- **管理**
我為新 NEM 文件網站的開發制定了路線圖，並接受了美術團隊的視覺指導。

  接下來一個月有兩位作者將全職投入教學製作，因此我們需要時間準備任務清單與工作批次安排。

總之，2月就是這樣：工作、工作、再工作，完全沒有娛樂。也就是沒有 XYM City。不過這其實對遊戲反而是好事！稍微拉開距離，讓我能以更新鮮的狀態重新審視。等我回到 XYM City 時，會帶著爆發性的能量，以及更多可分享的內容回來。

## 接下來的計畫

Zero 和我會持續進行 Symbol 文件衝刺，直到教學全部完成。

之後，他會轉去新的 NEM 文件，我會回到審閱工作……接著再投入 XYM City。

另外，新版 Symbol 錢包也進展順利，完成後我會立刻撰寫指南。這些比開發者教學簡單得多。

我還沒有完全脫身，但已經看見隧道盡頭的光了！而且還是霓虹色的。😎

感謝你讀到這裡。我們下個月再見！
