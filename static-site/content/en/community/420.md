XYMcomic is a series of Fully-On-Chain NFT manga that is released every two weeks in Japanese and English on [the official XYMcomic Twitter](https://twitter.com/xym_comic?s=20). The work is published as a Fully-On-Chain NFT using the Symbol blockchain and auctioned on [Twitter](https://twitter.com/xym_comic?s=20).

The comics, drawn by illustrator [Toujou-Mina 東城みな](https://toujoumina.com), are produced by [Atomos-Seed LLC](https://atomosseed.com) using [NFTDrive](https://nftdrive.net), a Fully-On-Chain NFT uploader developed using the Symbol blockchain.

* [Toujou-Mina 東城みな Twitter](https://twitter.com/toujoumina?s=20)　
* [Atomos-Seed LLC Twitter](https://twitter.com/atomos_seed?s=20)　
* [NFTDrive Twitter](https://twitter.com/nftdrive_jp?s=20)　


Fully on-chain NFT store all associated data directly on the blockchain, rather than the more common form of NFT where data such as images are stored on a file system outside the blockchain. This reduces the risk of loss or tampering with the data itself, and the data can be stored securely.
XYMcomic is a series of blockchain comic strips with the story centred around the NEM/Symbol community "XYM CITY".
To create the Fully On-chain NFT, a Fully On-chain NFT uploader "[NFTDrive](https://nftdrive.net)" is used, which means that the data can be stored almost permanently and the copyright can be clearly displayed.
  

This page is an archive of images from the comics that have been released. (Episodes 21-40)
