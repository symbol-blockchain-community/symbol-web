The alpha version of NFTDriveEditor, the world's first fully-on chain blockchain-based document editor, is now available on NFTDriveEX.

![NFTDriveEditor-1.png](../images/NFT_Drive_Editor_1_f58cdf3524.png)

The NFTDriveEditor allows users to create all types of documents, including various business texts, contracts, medical records and wills, which can be archived and stored the blockchain.

Announcements about the release of NFTDriveEditor can be found at the following URL for sharing the document created with NFTDriveEditor.
