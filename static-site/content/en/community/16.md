## NEM/Symbol-based Dapps and services introduction page.

### **Wallet**
#### `Symbol`
* [**Desktop wallet**](https://github.com/symbol/desktop-wallet/releases) Linux, Windows and Mac
* [**Mobile wallet** Android](https://play.google.com/store/apps/details?id=nem.group.symbol.wallet&hl=ja&gl=US) 
* [**Mobile wallet** iOS](https://apps.apple.com/us/app/symbol-wallet/id1528741845) 
* [**Mobile wallet AEM+** Android](https://play.google.com/store/apps/details?id=com.aem.plus) 
* [**Mobile wallet AEM+** iOS](https://apps.apple.com/jp/app/aem/id1607457067) 
AEM+ wallet is a multi-chain (XYM,XEM, BTC, ETH)mobile wallet that allows you to manage your crypto transactions on any device. Designed to store all your crypto wallets in one place — the palm of your hand. 
* [**Mobile wallet EXYM** Android](https://play.google.com/store/apps/details?id=com.ry0chang.EXYM) 
* [**Mobile wallet EXYM** iOS](https://apps.apple.com/jp/app/exym/id6449445030) 
EXYM is a new style cryptocurrency wallet application that makes it easy to enjoy digital art in everyday life. It allows users to easily create their own skin design wallets, creating a new experience that goes beyond the conventional use of NFTs. For art creators, it provides an opportunity to enjoy their own artwork as a new form of value.
* [**Mobile wallet CoinPocket** Android](https://play.google.com/store/apps/details?id=net.teritaris.coinpocket&hl=ja&gl=US)
* [**Mobile wallet CoinPocket** iOS](https://apps.apple.com/jp/app/coinpocket/id6479215775)
CoinPocket aims to be easy to use without the complexity of other crypto wallets.
* [**Browser wallet NFT-DriveEX**](https://chrome.google.com/webstore/detail/nftdriveex/oodjpaddnaphopckiphmgapkafegcghl)
NFTDriveEX is a p2p wallet that allows users to exchange NFTs, XYM and other mosaics from anywhere on the site.
[Google Chrome Extension Version](https://chromewebstore.google.com/detail/nftdriveex/oodjpaddnaphopckiphmgapkafegcghl?hl=ja)
[Web(Universal version)](https://nftdrive-ex.net)
* [**Browser wallet SSS(Safely Sign Symbol) Extension**](https://chrome.google.com/webstore/detail/sss-extension/llildiojemakefgnhhkmiiffonembcan)
Symbol Blockchain SSS Extension is a Google Chrome Extension that works with web applications that use the Symbol blockchain. This SSS application is designed to sign transactions on the Symbol blockchain, without the need to process a private key.
Although this app is an extension that allows you to securely sign transactions, it can also be used as a browser wallet.
* [**Browser wallet Ventus Wallet**](https://ventus-wallet.tk)
This is a Symbol web wallet that uses a Google Chrome Extension "SSS Extension". 
* [**Browser wallet One Plate**](https://oneplate.food-nft.io)
ONEPLATE is an easy-to-use tool for starting NFTs. It is designed to be easy to understand and use, with the aim of making it 'anyone can do it now'.

* [**QUEST**](https://quest-bc.com)
QUEST is a novel "contingency fee basis" tipping platform that build on block chain technologies.
“The quest” is an achievement you are aiming on the challenge.
Supporters who expecting your struggle journey and successful finale boosts vault of the quest by donation.
After you accomplish the quest, you’ll acquire the vault contents. If you don't make it... it'll be refunded to the supporter who donated.
But who judge the quest was succeeded or failed? It’s supporter’s vote.
QUEST main features are full automatic and safe processing that realized by block chain technologies.
